These adverbs have two forms:

| without -ly | with -ly |
| --- | --- |
| fair | fairly |
| free | freely |
| high | highly |
| late | lately |
| most | mostly |
| near | nearly |
| pretty | prettily |
| right | rightly |
| wrong | wrongly |
